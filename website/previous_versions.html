Park Observer

An iPad application for offline data collection

Brought to you by the Alaska Regional Office GIS Team


Links to all versions of Park Observer

Please install the current version. Previous versions, are provided in case you are have trouble with the latest release. In general downgrading is not recommended. If you are having trouble with the current release, please Contact us.

To check the version, open the Settings App and tap on Park Observer near the bottom of the list on the left. NOTE There is a bug in version 9.x of the Settings app - if the Park Observer settings page is mostly blank, kill the settings app (double click the home button and swipe up on the settings app) and try again.

All versions of Park Observer are signed with an expiring certificate, and will no longer run when the certificate expires.

To install go to https://akrgis.nps.gov/observer on your iPad, or follow these instructions from your PC.
