Links to all Versions of Park Observer

These previous versions, are provided in case you are having trouble with the latest release, and you wish to downgrade. In general this is not recommended. If you are having trouble with the current release, please Contact us.

To check the version, open the Settings App and tap on Park Observer near the bottom of the list on the left. NOTE There is a bug in the Settings app - if the Park Observer settings page is mostly blank, kill the settings app (double click the home button and swipe up on the settings app) and try again.

All versions of Park Observer are signed with an expiring certificate, and will no longer run when the certificate expires.
