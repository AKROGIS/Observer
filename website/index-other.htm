Park Observer

An iPad application for offline data collection

Brought to you by the Alaska Regional Office GIS Team



Getting Started

To use Park Observer, you will need an iPad with at least iOS 9 (most recent version is recommended).

Is your iPad is on a NPS WiFi?

Yes: Use the browser on your iPad to go to https://akrgis.nps.gov/Observer, and follow the instructions on that page.

No: Contact you local IT to see if it is possible to get your iPad on NPS WiFi. If not, then follow these instructions.

I don't know: Use the browser on your iPad to go to https://akrgis.nps.gov/Observer. If you get a web page, then you are on NPS WiFi.

Resources

  • Application Downloads

    Current and previous versions of the Park Observer application for download and offline installation.

  • Maps

    A list of ready to use basemaps.

  • Protocols

    A list of ready to use survey protocols. A protocol specifies the attributes to collect at each observation and other specific requirements for a particular type of survey.

  • Survey Results

    A list to data (and maps) collected form Park Observer. This list only contains data that has been Synced with the server on the iPad.

  • Creating Maps/Protocols

    Instructions on what to do if there is no protocol or basemap for your survey.
